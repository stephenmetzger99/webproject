Twitter Scraper

Description

Pulls tweets from a curated list of users I enjoy and displays them on an HTML page in the form of a news ticker.

It was written in python and uses the selenium webdriver library to automate the process. The prgram reads a text file listing twitter users, then tweets are scraped from twitter. After this they are stored in a JSON file, then a JavaScript function reads the JSON file. Lastly, it displays them on the webpage in a marquee.
