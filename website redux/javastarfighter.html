Java Starfighter
Game

Description

This is a project I started during my winter break between Fall 2017 and Spring 2018 College Semesters. I started this project to improve upon my programming skills and have worked on it here and there in my free time. The ships are inspired by Star Citizen.

GitHub

Concept

The game increments through rounds of enemies based on how long you've been in the game. Every second your score increases. The player must be weary of alien ships, flying lasers, and asteroids. Killed enemies will drop health points to restore ship health. Eventually, an alien kingship will spawn and you must defeat it.

Eventually, I would like to implement ship upgrades, saving, and score tracking, as well as a more fleshed out navigation system.
